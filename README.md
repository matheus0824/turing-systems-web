# Turing Systems - Tech Consultancy

> "From simple logic to grand solutions."

## Project Overview
Turing Systems is a software consultancy project inspired by the legacy of **Alan Turing**. The core concept connects the origins of computing with the expansion of the universe, showing how small, precise steps lead to massive growth.

This project transitioned from a SPA to a **multi-page architecture** to prioritize performance and SEO.

## Tech Stack
* **Frontend:** HTML5 & CSS3 (Responsive Design)
* **Logic:** JavaScript
* **Philosophy:** Universal evolution and Turing's logic

## Key Features
- **Multi-page Structure:** Better indexing and navigation.
- **Visual Identity:** Dark-themed UI inspired by the cosmos.
- **SEO Optimized:** Semantic tags for better search rankings.
